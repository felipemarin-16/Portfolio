# Portfolio

Personal portfolio website for Wilson Felipe Marin built with Next.js and Tailwind CSS.

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the portfolio.

## Build

```bash
npm run build
```

The site will be exported as static HTML in the `out` directory.
